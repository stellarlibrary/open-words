# 🔬 latency

🔊 /ˈleɪtənsi/<br>
🏷️ *noun | 名词*

---

## 📖 Meaning | 释义
**EN:** (n.) The state of existing but not yet being developed or manifest; the delay before a transfer of data begins following an instruction
**中文：** n. 潜伏，潜在；潜伏物；（数据传输的）延迟

---

## 🔗 Synonyms | 近义词
- dormancy（休眠）
- delay（延迟）
- inactivity（不活跃）
- quiescence（静止）

---

## 📌 Usage Tip | 用法提示
Use with: **low latency / high latency / network latency / latency period / reduce latency**

👉 Example: The engineering team worked tirelessly to reduce network latency for real-time communication applications.

---

## 📝 Examples | 例句

1. The virus may remain in a state of **latency** for years before clinical symptoms finally become apparent in the host.
   该病毒可能在潜伏状态下存留多年，然后临床症状最终在宿主体内显现。

2. Reducing **latency** in data transmission is critical for the successful deployment of autonomous vehicle navigation systems.
   降低数据传输延迟对于自动驾驶车辆导航系统的成功部署至关重要。

3. The researchers measured the **latency** between stimulus presentation and neural response to map brain processing pathways.
   研究人员测量了刺激呈现与神经反应之间的潜伏期，以绘制大脑处理通路。

---

## 🧩 Memory Trick | 记忆技巧
Think of "latency" as "LATE-ncy" — something that arrives "late" because it is hidden or delayed. Latent things show up late.
把"latency"想成"late（迟到的）+ ncy"——因为隐藏或延迟而"迟到"的东西。潜伏的事物总是姗姗来迟。
